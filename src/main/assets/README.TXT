About this mod
--------------------

This mod is a common logics module of mod that created by alalwww.
This mod has no additional features, but it is necessary in order to use the mod that created by alalwww.

この mod は alalwww によって作成されたMODの共通のロジックモジュールです。
導入することによる追加要素などはありませんが、alalwww によって作成された MOD を使用するために必要です。


Installation
--------------------

Put the "AwA-commons_x.x.x_x.x.x.x.jar" file to .minecraft/mods folder.

AwA-commons_x.x.x_x.x.x.x.jar ファイルを .minecraft/mods フォルダーに配置します。
